# Meu Portfólio — Adriano JR

Portfólio pessoal desenvolvido com **Next.js**, **React** e **Tailwind CSS**, apresentando minha trajetória como desenvolvedor de software focado em front-end.

## Sobre

Sou **Adriano Da Silva Dantas Junior**, estudante de Análise e Desenvolvimento de Sistemas na UNIFIP, com experiência prática em React, Next.js, Java e Spring Boot. Foco em criar interfaces modernas, responsivas e bem estruturadas, unindo boas práticas de código a uma boa experiência de uso.

## Destaque visual

A hero section traz um **crachá com física de corda em tempo real** (canvas + integração Verlet): a fita reage à gravidade, pode ser arrastada e solta livremente, estica de forma elástica até um limite (e então resiste) e "sobe" visualmente para fora do topo da página, como se estivesse pendurada em algo fora do site.

## Stack

- **UI:** React 19
- **Estilização:** Tailwind CSS 4
- **Linguagem:** TypeScript 5
- **Deploy:** Export estático (GitHub Pages)

## Como Rodar Localmente

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

O projeto usa `basePath: "/Meu-Portifolio"` (necessário para o deploy no GitHub Pages), então em desenvolvimento acesse:

[http://localhost:3000/Meu-Portifolio/](http://localhost:3000/Meu-Portifolio/)

## Scripts

| Comando | Descrição |
|---------|-----------|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera o build de produção (export estático) |
| `npm run start` | Inicia o servidor de produção |
